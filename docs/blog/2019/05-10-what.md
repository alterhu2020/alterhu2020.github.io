## ionic bottomsheet弹出带有表格的actionsheet?

全局样式中如下定义：

```
/*// 一半的弹出框*/
.half-modal {
  .modal-wrapper {
    height: 50%;
    top: 50%;
    position: absolute;
    display: block;
  }
}

```
