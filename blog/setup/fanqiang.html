# 操作步骤

# Windows

注： 你也可以使用 IPsec/L2TP 模式 连接，无需安装额外的软件。

  1. 下载并安装免费的 Shrew Soft VPN 客户端。
    注： 该 VPN 客户端支持 Windows 2K/XP/Vista/7/8 系统。

  2. 安装提示操作安装完成即可。

  3. 单击开始菜单 -> 所有程序 -> ShrewSoft VPN Client -> VPN Access Manager

  4. 单击工具栏中的 Add (+) 按钮。

  5. 在 Host Name or IP Address 字段中输入你的 VPN 服务器 IP： 34.69.204.193

  6. 单击 Authentication 选项卡，从 Authentication Method 下拉菜单中选择 Mutual PSK + XAuth。

  7. 在 Local Identity 子选项卡中，从 Identification Type 下拉菜单中选择 IP Address。

  8. 单击 Credentials 子选项卡，并在 Pre Shared Key 字段中输入你的 VPN IPsec PSK： 55e76a3d2285ee1264998ue9597b6f383a978ed6b

  9. 单击 Phase 1 选项卡，从 Exchange Type 下拉菜单中选择 main。

  10. 单击 Phase 2 选项卡，从 HMAC Algorithm 下拉菜单中选择 sha1。

  11. 单击 Save 保存 VPN 连接的详细信息。

  12. 选择新添加的 VPN 连接。单击工具栏中的 Connect 按钮。

  13. 在 Username 字段中输入你的 VPN 用户名: alterhu2020

  14. 在 Password 字段中输入你的 VPN 密码: 1235674378778

  15. 单击 Connect
