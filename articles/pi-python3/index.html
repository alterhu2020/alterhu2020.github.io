[Linux环境搭建] 如何完整卸载python2/3 在debain上

2018-07-12 21:40:26   By Walter Hu ,   

本文出自PingBook Blog,转载请务必保留文章链接

卸载 remove python3¶

$ sudo apt-get remove 'python3.*' 其实这里的python的包是: `python3.5-minimal`,`python3.5-venv`

安装 install python3.7¶

因为默认的debian仓库里面是python3.5,所以如果需要安装最新的python,按照如下步骤进行安装, 参考文档: https://unix.stackexchange.com/questions/332641/how-to-install-python-3-6

需要用到的命令如下: “` 安装依赖包 $ sudo apt-get install -y make build-essential libssl-dev zlib1g-dev libbz2-dev \ libreadline-dev libsqlite3-dev wget curl llvm libncurses5-dev libncursesw5-dev \ xz-utils tk-dev libffi-dev $ sudo wget https://www.python.org/ftp/python/3.7.0/Python-3.7.0.tgz $ tar xvf Python-3.7.0.tgz // 下面的步骤安装会跑测试来判定机器的性能,从而进行性能的提高 $ ./configure –enable-optimizations –with-ensurepip=install $ make -j8 $ sudo make altinstall $ sudo update-alternatives –install /usr/bin/python python /usr/bin/python3.7 50 $ sudo update-alternatives –install /usr/bin/python3 python3 /usr/bin/python3.7 50

“`

$ pip install –user pipenv /home/pi/.local/bin $ python -m site –user-base $ sudo nano ~/.profile export PATH=”$HOME/.local/bin:$PATH” $ sudo source ~/.profile $ pipenv –update
