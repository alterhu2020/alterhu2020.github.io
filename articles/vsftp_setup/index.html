How to build the ftp system

2018-08-10 11:41:48   By Walter Hu ,   

本文出自PingBook Blog,转载请务必保留文章链接

起因¶

需要搭建一个文件共享的服务器,这样可以方便的进行文件的共享和存取. 参考博客: https://www.conum.cn/os/linux/210.html

1. 安装vsftpd
$ apt-get install vsftpd 
安装完成后,会新建一个用户组ftp和对应的用户名ftp,通过下面的命令查看他的配置
$ sudo nano /etc/passwd 
已经设置成了/srv/ftp目录,这里把它改成 /ftp目录
$ aptitude search db | grep util
$ aptitude install db5.3-util
$ db5.3_load -T -t hash -f /etc/vsftpd/login.txt /etc/vsftpd/vsftpd_login.db
$ chmod 600 /etc/vsftpd/ftpusers.db
2. 查看log
$ tail -f -n 50 /var/log/auth.log
